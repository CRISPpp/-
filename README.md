# 贪吃坤小游戏，android开发，真爱粉力作
# 作者我是个ikun，正好收集了坤坤的素材，然后写了个安卓版的贪吃坤来应援坤坤，素材包括音乐、照片均来自于网络，和作者本人无关。
主要的功能包括可视化的图形界面，同时能记录历史数据，如果历史记录新高，会出现新的结算界面和语言，同时可以调节游戏的速度，坤坤的头部碰到身体或者墙上则结束，并跳转到结算界面，结算界面则显示本轮的分数以及是否突破最高分和重开按钮。
